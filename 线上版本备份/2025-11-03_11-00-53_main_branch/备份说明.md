## 备份内容

这个备份包含了 Copus Network 生产环境的最新代码，主要功能包括：

### 核心功能
- **x402 支付协议**: 基于 ERC-3009 的无gas费 USDC 支付系统
- **钱包认证**: 支持 MetaMask 和 Coinbase Wallet 登录
- **内容解锁**: 付费内容的解锁和访问系统
- **社交功能**: 用户关注、点赞、收藏等

### 技术栈
- React 18.2 + TypeScript 5.9
- Vite 6.0 构建工具
- Tailwind CSS 3.4 + Radix UI
- TanStack Query (React Query) 状态管理
- Web3 钱包集成

### 备份文件
- `src/` - 完整源代码目录
- `package.json` - 项目依赖配置
- `CLAUDE.md` - 开发文档和架构说明
- `X402_PAYMENT.md` - x402 支付协议文档
- `tailwind.config.js` - Tailwind CSS 配置
- `vite.config.ts` - Vite 构建配置

### 重要特性
1. **x402 支付协议**
   - 基于 Base Sepolia 网络的 USDC 支付
   - ERC-3009 TransferWithAuthorization 无gas费交易
   - 2-3秒快速支付确认

2. **钱包集成**
   - MetaMask 和 Coinbase Wallet 支持
   - 自动账户创建和登录
   - 多钱包环境智能检测

3. **内容系统**
   - 付费内容锁定和解锁
   - Arweave 永久存储集成
   - 社交互动功能

## 使用说明

要运行此版本的代码：

```bash
npm install
npm run dev
```

访问开发环境: http://localhost:5177

## 注意事项

- 这是生产环境的代码备份
- 包含完整的 x402 支付功能
- 需要配置相应的环境变量才能正常运行
- Base Sepolia 网络测试需要测试 USDC
